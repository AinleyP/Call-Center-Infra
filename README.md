# Call Center Application

### Purpose
A call center that takes user's dialogue with chat, transcribes their speech to readable text, and notifies the specific person through email with the attached transcription.

### Tools:
* AWS Connect
* S3 Bucket
* Step Functions
* AWS Lambda
* AWS Transcribe
